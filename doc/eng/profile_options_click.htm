PROFILE OPTIONS

CLICK


 

- Dwell click enabled

 

If selected enables eViacam controlling the pointer.

 

 

DWELL CLICK

 

 

- Dwell time

 

Is the lapse of time before the generation of the click after stopping the pointer on the desired location. The units are tenths.

 

- Dwell area

 

To generate the click it is not necessary to stop the pointer completely. EViacam allows a small movement margin around the exact place where you want to click. If this value increases the margin around the place where you want to click also increases. The units are pixels of the screen.

 

- Allow consecutive clicks

 

It allows to execute several clicks over the same location while the pointer is stopped.

 

- Beep on click

 

If selected eViacam will play a short sound (beep) every time a click is generated.

 

- Allow visual alerts

 

It provides a visual information to the user about the dwell click progress end the direction in gesture click mode.

 

- Enable click with gestures

 

If selected the user chooses the type of click moving up, down, left or right.

 

 

CLICK WINDOW

 

 

- Open click window at startup

 

If selected the Click Bar will be automatically open at startup.

 

- Show click window

 

Opens the click bar.

 

- Design

 

Selects the click bar design. Normal pattern shows each action in a customized icon. The thin pattern takes less space in the screen and each action is represented by a button.

 

- Behaviour

 

Allows to select how to choose the action to perform. Normal mode requires a clic on the desired icon to select an action of the Click Bar. Fast mode avoids the one click delay to select an action of the Click Bar. Now the selection it is done moving the pointer over the desired action selection button. To set the Predetermined action it's only needed one click.

 

See click bar

 

GESTURE CLICK

 

 

- Gesture time

 

Is the lapse of time before the generation of the click after indicating the desired direction through a nod. The units are tenths.

 

- Move left

 

Select the action to perform with a left nod.

 

- Move right

 

Select the action to perform with a right nod.

 

- Move top

 

Select the action to perform with a top nod.

 

- Move bottom

 

Select the action to perform with a bottom nod.

 

See click modes
