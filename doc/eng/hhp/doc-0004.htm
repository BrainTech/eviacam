WORKING ENVIRONMENT


MAIN ELEMENTS


The working environment of the eViacam is composed of three basic elements.

 

  • The click bar.

  • The main window.

  • The task bar icon.

 

 

MAIN WINDOW


It is the most important element. Through the menu and the toolbar you can accede to all options of the application.

 

 

In turn, the main window shows the user's image. Check that this fulfills the requirements for a correct operation (see web camera's key factors). Otherwise, make the pertinent image adjustments.

 

In the live video area the Active Zone appears. This is the blue rectangle shown in the figure. The information picked up in this area is the key for the operation of the application. Please refer to the configuration wizard for more information on how to configure properly the Active Zone.

 

CLICK BAR


The Click Bar is the main complement for the selection and execution of clicks in dwell click mode. It is a graphical element and let the user change the kind of click to generate (left, right, drag & drop and double click).

 

 

TASK BAR ICON


Let the user to open the main window when this is minimized.
