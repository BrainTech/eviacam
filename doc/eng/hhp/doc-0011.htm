CLICK BAR


The Click Bar is a small bar that appears docked in the top of the screen. It allows to choose the different possible actions (left button, right button, drag & drop and double click) when a click is generated.

 

 

The Click Bar is only compatible with the Dwell Click and has two different action selection mode.

 

Normal mode

 

To select the action that will be generated the next time (after a click), simply click one of the click selection buttons on the Click Bar. To set the predetermined action simply make two consecutive clicks in one of the click selection buttons of the Click Bar.

 

Fast mode

 

This mode avoids the one click delay to select an action of the Click Bar. Now the selection it is done moving the pointer over the desired action selection button. To set the Predetermined action it's only needed one click.

 

 

BUTTON MEANING

 

No click

 

It allows to disable the generation of the click. It is a useful function to calmly see a page, a document, etc. It happens that if we don't control the position of the pointer this can generate undesired clicks. To avoid this press the 'No click' button and the click generation will be disabled. To enable click generation press the 'No click' button again.

 

Left

 

Identical function that the left button of a conventional mouse.

 

Right

 

Identical function that the right button of a conventional mouse, this click opens a contextual menu for a folder or file for example.

 

Drag

 

Identical function that maintaining pressed the left button (first click) of a conventional mouse and release it later (second click). This action allows to arrange the desktop icons or make the selection of a text for example.

 

Double click

 

Identical function that a double pulsation with the left button of a conventional mouse.

 

Minimize window icon

 

Minimize or restore the main window of eViacam.
