WIZARD

DESCRIPTION


The Configuration Wizard will allow you to configure quick and simply the main options to begin to work and to get the maximum performance of eViacam.

 

 

WIZARD STEPS


1. Start

 

This is the welcome window. Click next button to continue the wizard.

 

 

2. Camera setup

 

Detects the web camera model, show the capture speed (FPS) and allows you to open the camera settings window to configure all the camera controls.

 

 

3. Pointer calibration (warning)

 

In this step, you should choose if you want to perform the automatic calibration method. See automatic calibration method

 

 

4. Pointer calibration (fine tuning)

 

After the automatic calibration (or if you do not want to perform it), you may adjust the pointer speed in horizontal and vertical axis and test the pointer motion through the 'Enable motion' button.

 

 

5. Click mode (warning)

 

This window informs that in the next step eViacam will be enabled to test the different ways of click. Click the next button to continue the wizard.

 

 

6. Click mode

 

Select a click mode and, through the 'Test' button, you can train the different types of click.

 

 

7. The end

 

In this step, the eViacam wizard has finished. Here you can choose three different options:

- Run the wizard when eViacam starts.

- Start eViacam when the user logon.

- Enable motion when eViacam starts.
