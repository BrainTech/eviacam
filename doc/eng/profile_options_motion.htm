PROFILE OPTIONS

MOTION


These values should be properly adjusted. We recommend checking the result each adjustment step. Good settings will make eViacam quicker and precise.

 

- X axis speed

 

Pointer speed of the horizontal direction (X axis).

 

- Y axis speed

 

Pointer speed of the vertical direction (Y axis).

 

- Acceleration

 

The acceleration causes that the pointer advances more when you move the head more quickly.

 

- Smoothness

 

Filter small tremors. A greater value, a softer point motion.

 

- Easy stop

 

This option facilitates the task of stopping the pointer completely. The units are pixels of the screen.

 

- Enabled at startup

 

If selected eViacam will be automatically enabled controlling the pointer.

 

- Calibrate motion

 

This option opens the dialog to calibrate the pointer speed automatically. See automatic calibration method
