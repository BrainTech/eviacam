CONFIGURATION OPTIONS

GENERAL OPTIONS


 

 

RUN AT STARTUP

 

- Start eViacam at user logon

 

If selected eViacam will start automatically at system startup.

 

 

PROFILE MANAGEMENT

 

- Select profile

 

Allows to choose a specific profile or the default profile.

 

- Add profile

 

Adds a new profile into the eViacam.

 

- Delete profile

 

Removes a profile from the eViacam.

 

 

SELECTED CAMERA

 

- Settings

 

Opens a window where you may modify all the camera controls.

 

- Change

 

Allows to change the selected camera.

 

 

LANGUAGE

 

Allows to choose the language of the eViacam.
