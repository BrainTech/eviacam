AUTOMATIC CALIBRATION METHOD

DESCRIPTION


The automatic calibration method allows you to adjust the pointer speed with only moving your head.

 

 

STEPS


It is important adjust the active zone before starting the automatic calibration method.

 

The first window of this method asks the user to move his head horizontally to set the pointer speed of the X axis.

 

 

When the system has set this parameter, it asks the user to move his head vertically to adjust the pointer speed of the Y axis.

 

 

Finally, if the system has detected enough motion, the user may test the pointer speed and accept, cancel or repeat the procedure.
