Configuration options

eViacam has a complete configuration options panel that allows to customize the behavior of the application.

The configuration panel is divided into two blocks. The General options panel, where you can adjust the camera settings and work with profiles, and the Profile options panel, where you can configure all the application features that best suits your needs.


General options



PROFILE MANAGEMENT


- Select profile

Allows to choose a specific profile or the default profile.


- Add profile

Adds a new profile using the configuration of the current one.


- Delete profile

Deletes a configuration profile.



CAMERA


- Settings button

Opens a window where you can modify all the camera controls.


- Change

Allows to change the selected camera.



LANGUAGE


Allows to choose the language of the eViacam.


Profile options

Motion


These values should be properly adjusted. We recommend to check the result after each adjustment step.



- X axis speed

Pointer speed of the horizontal direction (X axis).


- Y axis speed

Pointer speed of the vertical direction (Y axis).


- Acceleration

The acceleration causes that the pointer advances more when you move the head more quickly.


- Smoothness

Filter small tremors. A greater value, a softer point motion.


- Motion threshold

This option facilitates the task of stopping the pointer completely. The units are pixels of the screen.


- Assisted calibration

This button starts the automatic calibration method. See automatic calibration method for further details.



Workspace



- Enable workspace limit

If selected, the pointer can only move within the bounded area.


- Top

Top limit of the workspace (% screen height)


- Left

Left limit of the workspace (% screen width)


- Right

Right limit of the workspace (% screen width)


- Bottom

Bottom limit of the workspace (% screen height)


- Allow wrap the pointer

If selected, the pointer wraps to the opposite edge instead of stopping when reaches the end of the workspace.



Click



- Enable dwell click

If checked allows eViacam to generate clicks.


- Dwell time

Time to wait (deciseconds) before generating a click action.


- Dwell area

To generate the click it is not necessary to stop the pointer completely. eViacam allows a small movement margin around the exact place where you want to click. If this value increases the margin around the place where you want to click also increases. The units are pixels of the screen.


- Allow consecutive clicks

It allows to execute several clicks over the same location while the pointer is stopped.


- Beep on click

If selected eViacam will play a short sound (beep) every time a click is generated.


- Allow visual alerts

It provides visual feedback about the dwell click progress end the direction in gesture click mode.



CLICK WINDOW


- Show click window

Opens the Click Window.


- Design

Selects the Click Window design. Normal design uses buttons with icons whereas Thin design uses text based buttons to save space.




- Behaviour

Allows to select how to choose the action to perform. Normal mode requires one clic to select an action and two clics to set the predetermined action. Fast mode allows to select the desired action by hovering the pointer and to select the predetermined action making one click. Thus, Fast mode saves one click.


- Location

It allows to select the click window orientation. It may be docked on the top, bottom, left or right of the screen, floating horizontal, or floating vertical. If the window click overlaps with another bar, a warning message appears. You can decide whether to keep or modify the orientation of the window.


- Autohide

If selected, the docked click window will be visible only if the pointer stands over it.


See Click Bar



GESTURE CLICK


- Enable gesture click

If selected the user chooses the type of click by moving up, down, left or right. See click modes for further details.


- Move left

Select the action to perform with a left nod.


- Move right

Select the action to perform with a right nod.


- Move top

Select the action to perform with a top nod.


- Move bottom

Select the action to perform with a bottom nod.


See click modes



Hotkeys



A set of eViacam functions may be configured to be managed through hotkeys. To enable one of them, you should tick the appropriate check box. Press the 'Set key' button to configure the desired key.



Advanced




STARTUP


- Start eViacam at user logon

If selected eViacam will start automatically at system startup. Not available on Windows.


- Enable eViacam at startup

If selected eViacam will be automatically enabled once started.



ON-SCREEN KEYBOARD COMMAND


Allows to select the command to open the on-screen keyboard.



FACE LOCALIZATION


- Locate face automatically

If checked, eViacam tries to detect the user's face to automatically centre the tracking area.


NOTE: Sometimes background colours similar to the skin colour can confuse eViacam. In case of trouble, disable this option and center the box manually.


- CPU Usage

Sets the amount of CPU used by the face detection system. Higher values cause a better face detection efficiency but increases the CPU performance whereas lower values decrease the CPU performance at the expense of face detection efficiency.


- Stop the pointer when face not detected

If selected, eViacam will be disabled if face is not detected. You may set the time without face before stop the pointer.
