Working environment


The working environment of the eViacam is composed by three basic elements.

  • The main window.

  • The Click Window.

  • The task bar icon.



MAIN WINDOW



The main window shows the camera image and provides access to all commands and configuration options. Once eViacam is started, this window opens and begins streaming live video. Make sure that image quality and capture speed fulfill the required standards for a correct operation (see About webcams for further details). In the live video area the Tracking Area (blue blox) is depicted. This is the blue rectangle shown in the screenshot. The motion picked up inside this area is the key for the operation of eViacam. Please refer to the configuration wizard for more information on how to properly configure the Tracking Area.



CLICK WINDOW



The Click Window is a complement for selecting amongst the different kinds of click (left, right, drag & drop and double click) in Dwell Click mode. Please, refer to Click Window help page for further details.



TASK BAR ICON



Allows to restore or minimize the main window.
