Configuration wizard

The Configuration Wizard will help you to configure the most common options of eViacam. The Configuration Wizard is automatically open the first time eViacam is started. You can open it through the main window menu.



Wizard steps


1. START


This is the welcome screen. Click next button to continue the wizard.




2. CAMERA SETUP


Detects the web camera model, shows the capture speed (fps) and allows you to open the camera settings window.




3. TRACKER SETUP


The information picked up inside the blue rectangle (tracking area) is the key for the operation of eViacam. That is why it is very important to adjust the tracking area to set it around the user's face.


You may enable the face detection system to detect the face automatically or adjust the tracking area manually. In automatically mode, you can set a no-face-detected time to disable eViacam.




4. POINTER CALIBRATION 1/2


In this step, you should choose if you want to perform the automatic calibration method to adjust the pointer speed. See automatic calibration method for further details.




5. POINTER CALIBRATION 2/2


In this screen you can manualy tune the pointer speed for both axis. To test the configuration, the 'Enable motion' is provided.




6. CLICK MODE 1/2


This screen explains the differents methods for clicking. Note that some options may not be available for all platforms.




7. CLICK MODE 2/2


This screen lets to choose the desired click method and to test it using the test button.




8. FINAL STEP
