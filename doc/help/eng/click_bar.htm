Click Window

The Click Window is a small bar that appears docked in the top of the screen. It allows to choose the action (left button, right button, drag & drop and double click) to be generated when clicking.

The Click Window is only compatible with the Dwell Click mode and has two different selection modes.

Normal mode

To select the action that will be generated the next time, simply click one of the buttons. To set the predetermined action simply make two consecutive clicks on the same button.

Fast mode

This mode avoids the one click delay to select an action of the Click Window. To select an action, just hover the pointer over the desired button. To predetermine an action, click once the desired button.

Button meaning

No click

It allows to disable click generation. It is a useful for reading a page, document, etc. without being disturbed by undesired click actions. To disable click generation press the 'No click' button. To enable click generation press again the 'No click'.

Left

Identical function as the left button of a conventional mouse.

Right

Identical function as the right button of a conventional mouse. This click opens a contextual menu for a folder or file, for instance.

Drag

Identical function as maintaining pressed the left button (first click) of a conventional mouse and releaseing it later (second click). This action allows to perform the drag and drop function.

Double click

Identical function as pressing twice the left button of a conventional mouse.

Minimize window icon

Minimize or restore the main window of eViacam.
