Introducció

Enable Viacam (eViacam) és un programa que substitueix la funcionalitat del ratolí permetent moure el punter a partir del moviment del cap. Funciona en un ordinador PC equipat amb una càmera web, sense elements addicionals. eViacam està basat en el programa Ratolí Facial guardonat amb diversos premis.

Amb Enable Viacam, vostè serà capaç de controlar l'ordinador sense utilitzar les mans. Per a això utilitza una càmera web estàndard per a seguir el moviment del seu rostre i controlar amb ell el punter del ratolí. També proporciona diversos mètodes per fer clic.

Principals característiques i avantatges

  • Mans lliures: Control de l'ordinador mitjançant intuïtius moviments del cap. No hi ha necessitat d'usar les mans, el punter simplement obeeix!
  • Sense cables: Funciona amb qualsevol webcam de qualitat mitjana o alta. Sense cables, sense enganxines, no requereix maquinari addicional.
  • Simplicitat: Instal·lació simple i ús intuïtiu. L'assistent de configuració integrat l'ajuda a començar de seguida. L'entrenament necessari es redueix al mínim.
  • Personalització: La velocitat del punter, l'acceleració i suavitzat, el temps de detenció, i altres variables poden ser configurades per ajustar-se a les necessitats de l'usuari.
  • Autonomia: Enable Viacam s'ha dissenyat específicament per eliminar la necessitat d'assistència una vegada instal·lat.

Plataformes suportades

  • GNU / Linux
  • Windows 2000/XP/Vista/7

NOTA: en l'actualitat, algunes funcions només estan disponibles per a GNU / Linux.
