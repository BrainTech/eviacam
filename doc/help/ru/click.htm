Режимы

Автоматический клик

Этот режим установлен по умолчанию. Клики генерируются автоматически, когда вы останавливаете курсор в нужном месте. Виды кликов можно выбирать в панели. По умолчанию установлен клик левой кнопкой. Смотрите панель.

Если визуальный вариант оповещения включен, индикатор рядом с указателем показывает обратный отсчет времени (эта опция не доступна для Windows)

Жесты

Этот режим позволяет Вам выбрать необходимый клик с помощью жестов головы. Для того чтобы сделать клик, остановите указатель на некоторое время в нужном месте. Затем переместите голову в одно из четырех основных направлений (вверх, вниз, влево или вправо), снова остановите и клик связанный с этим направлением будет сгенерирован автоматически.

Вы можете изменить клики, связанные с каждым направлением: Инструменты -> Настройки -> Клик

Если включен визуальный вариант оповещения, индикатор рядом с указателем покажет обратный отсчет, и при перемещении головы в желаемом направлении, на экране будет нарисована линия.

ПРИМЕЧАНИЕ: При использовании этого режима, рекомендуется включить визуальные предупреждения.
