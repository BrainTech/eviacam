Введение

Enable Viacam (eViacam) является свободным программным обеспечением. Заменяет мышь, перемещает курсор при движении головой. Работает на стандартном ПК, оснащенным веб-камерой. Никакого дополнительного оборудования не требуется. На основе, отмеченного наградами ПО Facial Mouse.

С eViacam, вы сможете управлять компьютером без помощи рук. eViacam использует стандартную веб-камеру для отслеживания движения лица и соответственно перемещает указатель мыши. Она также предоставляет несколько методов для нажатия(кликов).

Основные особенности и преимущества

  • Свободные руки: Управление компьютером с интуитивно понятными движениями головы. Нет необходимости использовать руки, указатель будет слушаться
  • Неприхотливость: Работает с любой веб-камерой хорошего качества. Никаких кабелей, дополнительного оборудования не требуется.
  • Лёгкость и простота: Простая установка, интуитивность использования. Встроенный мастер настройки легко поможет вам начать работу. Обучение сведено к минимуму.
  • Настраиваемость: Скорость указателя, ускорение движения и плавность, время задержки, и многие другие параметры могут быть скорректированы в соответствии с потребностями пользователя.
  • Автономность: Специально разработана для устранения необходимости в помощи после установки.

Поддерживаемые платформы

  • GNU/Линукс
  • Виндовс 2000/XP/Vista/7

Примечание: в настоящее время, некоторые функции доступны только для GNU / Linux систем.
