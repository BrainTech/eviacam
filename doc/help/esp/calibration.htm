MÉTODO DE CALIBRACIÓN AUTOMÁTICO

DESCRIPCIÓN


El método de calibración automático permite ajustar la velocidad del puntero utilizando solamente movimientos con la cabeza.

 

 

PASOS


Es importante ajustar la zona activa antes de empezar el método de calibración automático.

 

La primera ventana pide al usuario que mueva la cabeza horizontalmente para calcular la velocidad en el eje de las X.

 

 

Cuando el sistema haya calculado dicha velocidad, le pedirá mover la cabeza verticalmente para calcular la velocidad en el eje de las Y.

 

 

Finalmente, si el sistema ha detectado suficiente movimiento, podrá probar la velocidad del movimiento y confirmar si acepta, cancela o desea repetir el procedimiento.
