Método de calibración automático

El método de calibración automático permite ajustar la velocidad del puntero utilizando solamente movimientos de la cabeza.

Pasos

Es importante haber ajustado el área de seguimiento antes de empezar el método de calibración automático.

La primera pantalla pide al usuario que mueva la cabeza horizontalmente para calcular la velocidad en el eje de las X.

Cuando el sistema haya calculado dicha velocidad, le pedirá mover la cabeza verticalmente para calcular la velocidad en el eje de las Y.

Finalmente, si el sistema ha detectado suficiente movimiento, podrá probar la velocidad del movimiento y confirmar si acepta, cancela o desea repetir el procedimiento.
