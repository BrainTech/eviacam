Allgemeine Beschreibung

Enable Viacam (alias eViacam) ist eine Software zur alternativen Maussteuerung per Kopf. Dies funktioniert auf einem gewöhnlichen Standard-PC mit einer Webkamera. Es ist keine zusätzliche Hardware erforderlich. Basierend auf der preisgekrönten Software "Facial Mouse".

Mit eViacam ist es möglich den Computer ohne Hände zu bedienen. eViacam verwendet dazu eine normale Webkamera die die Kopfbewegungen erkennt und damit den Mauszeiger steuert. Es bietet auch unterschiedliche Möglichkeiten zum Klicken an.

Hauptmerkmale und Vorteile

  • Ohne Hände: Steuere den Computer mit intuitiven Kopfbewegungen. Dazu sind keine Hände erforderlich - der Mauszeiger wird dir gehorchen!

  • Anspruchslos: Funktioniert mit jeder guten Webkamera. Keine zusätzliche Hardware erforderlich.

  • Leicht und einfach: Einfache Installation, intuitive Bedienung. Durch den integrierten Konfigurationsassistent ist ein einfacher Start möglich. Das Training ist dadurch auf ein Minimum geschrupft.

  • Anpassbar: Mauszeiger Geschwindigkeit, Beschleunigung und Genauigkeit, Wartezeit und viele andere Parameter können auf die jeweiligen Bedürfnisse des Benutzers angepasst werden.

  • Unabhängig: Dieses Produkt wurde speziell entworfen um nach der Installation ohne Hilfe einer zweiten Person bedient zu werden.

Unterstützte Betriebsysteme

  • GNU/Linux

  • Windows 2000/XP/Vista/7

Beachte: Momentan sind manche Eigenschaften nur in der GNU/Linux Version verfügbar.
