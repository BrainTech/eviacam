Modos de hacer clic

Clic por espera

Este es el método por defecto de la aplicación para generar clics. Para realizar el clic, detener el puntero unos instantes en el área donde desee clicar. Después del tiempo preestablecido, el clic se realizará automáticamente. La acción a realizar por defecto es el clic izquierdo. Utilice la ventana de clic para seleccionar una acción diferente. Ver Ventana del clic.

Si la opción de alertas visuales está activada, aparecerá una barra de progreso al lado del puntero que mostrará el contador. No disponible para Windows.

Clic por gestos

Este método de generación de clic permite al usuario seleccionar la acción a realizar moviendo solamente la cabeza. Para generar un clic, detener el puntero en el área donde desee clicar. Después del tiempo preestablecido, mover la cabeza hacia la dirección asociada a la acción deseada. Cuando haya terminado el tiempo de espera de gesto se realizará el clic.

Puede modificar las acciones asociadas a cada movimiento a través de: Configuración -> Opciones -> Clic

Si la opción de alertas visuales está activada, aparecerá una barra de progreso junto al puntero para indicar los tiempos de espera. Cuando usted mueva la cabeza para indicar la acción que desea realizar, aparecerá una línea que seguirá al puntero desde el area donde se realizará el clic.

NOTA: Recomendamos activar la opción de alertas visuales para trabajar con este método de generación de clics.
