Entorno de trabajo

El entorno de trabajo de Enable Viacam se compone de 3 elementos básicos en pantalla.

  • Ventana principal.
  • Ventana del clic.
  • Icono de la barra de tareas.

Ventana principal

Es el elemento más importante y el que permite acceder a todas las opciones de configuración de la aplicación, activar, desactivar o cerrar la aplicación, etc. Utilize los íconos de la barra de herramientas o los menús para ello. Asegúrese de que la calidad de imagen y velocidad de captura cumplen con las normas requeridas para un correcto funcionamiento (ver Sobre webcams para más detalles). El rectangulo azul que aparece en el interior del cuadro de captura delimita la zona de seguimiento de movimiento y por ello es imprescindible ajustar correctamente esta zona de forma que el rostro del usuario aparezca en su interior. Consulte la sección asistente de configuración para más información acerca de como configurar el área de seguimiento.

Ventana del clic

La Ventana del clic es el complemento principal para la selección y ejecución del clic. Permite escoger entre las distintas modalidades de clic además de proporcionar una acceso directo para minimizar o restaurar la ventana principal cuando se utiliza el modo de clic por espera. Consulte Ventana del clic para más detalles.
