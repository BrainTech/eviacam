Introdución

Enable Viacam (eViacam) é un programa de computador que substitúe a funcionalidade do rato permitindo mover o punteiro a partires do movemento da cabeza. Funciona nun computador PC equipado cunha cámara web, sen elementos adicionais. eViacam está baseado no programa Rato facial galardoado con varios premios.

Con Enable Viacam, vostede será quen de controlar o computador sen empregar as mans. Para elo utiliza unha cámara web estándar para seguir o movemento da súa faciana e controlar con el o punteiro do rato. Tamén fornece varios métodos para premer (facer clic).

Principais características e vantaxes

  • Mans libres: Control do computador mediante movementos intuitivos da cabeza. Non sendo preciso empregar as mans, o punteiro simplemente obedece!
  • Sen fíos: Funciona con calquera webcam de calidade media ou alta. Sen fíos, sen adhesivos, non require hardware adicional.
  • Simpleza: Instalación simple e uso intuitivo. O asistente de configuración integrado axúdalle a comezar deseguida. O adestramento necesario redúcese ao mínimo.
  • Personalización: A velocidade do punteiro, a aceleración e suavizado, o tempo de detención, e outras variábeis poden ser configuradas para axustarse ás necesidades do usuario.
  • Autonomía: Enable Viacam foi deseñado especificamente para eliminar a necesidade de asistencia unha vez instalado.

Plataformas compatíbeis

  • GNU/Linux
  • Windows 2000/XP/Vista/7

NOTA: na actualidade, algunhas funcións só están dispoñíbeis para GNU/Linux.
